# Code Examples

Production-ready fraud detection code examples and implementations.

## Available Examples

### Basic Fraud Detection
- `basic_fraud_detector.py` - Complete fraud detector with XGBoost
- `feature_engineering.py` - Custom feature engineering pipeline

### Graph-Based Detection
- `graph_fraud_detection.py` - GNN-based fraud detection
- `graph_builder.py` - Transaction graph construction utilities

### Production APIs
- `real_time_api.py` - FastAPI fraud detection endpoint
- `mlops_pipeline.py` - MLflow training pipeline

## Quick Start
```bash
pip install -r requirements.txt
python basic_fraud_detector.py
```
